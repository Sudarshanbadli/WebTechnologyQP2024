Section A (2 Marks Each)

1. Define Internet.

The Internet is a global network of interconnected computers and servers that communicate using standardized protocols to share and exchange data, enabling access to information, communication, and services worldwide.

Example: Accessing www.google.com uses the Internet to fetch the webpage via HTTP.

2. What is Web Protocol? Give example.

A web protocol is a set of rules for data communication over the Internet.

Example: HTTP (HyperText Transfer Protocol) is used for transferring web pages. Visiting http://example.com uses HTTP to fetch the page.

3. Which tag is used to embed an image and audio in HTML?

- Image: The <img> tag, e.g., <img src="photo.jpg" alt="A beautiful sunset">.

- Audio: The <audio> tag, e.g., <audio src="song.mp3" controls></audio>.

4. Define HTML Element. Give example.

An HTML element consists of a start tag, content, and an end tag, defining webpage structure.

Example: <p class="intro">Welcome to my website!</p> â€“ <p> is the start tag, class="intro" is an attribute, and </p> is the end tag.

5. Name any two key features of XML.

- Extensibility: Allows custom tags, e.g., <product><name>Laptop</name></product>.

- Platform-Independence: Works across systems, e.g., an XML file can be read on both Windows and Android.

6. Write the Syntax of CSS.

CSS syntax: selector { property: value; }

Example: h1 { color: red; font-size: 24px; }

7. Give two uses of JavaScript in Web development.

- Interactivity: Enables form validation, e.g., checking if a username is empty before submission.

- Dynamic Updates: Modifies content without reloading, e.g., updating a counter when a button is clicked.

8. What is the use of try ... Catch Statement in JavaScript?

The try...catch statement handles errors by executing code in the try block and catching errors in the catch block.

Example:

try {
    let result = undefinedVariable;
} catch (error) {
    console.log("An error occurred: " + error.message);
}
9. Define Event in JavaScript.

An event is an action (e.g., click, keypress) that triggers a JavaScript function using event handlers like onclick.

Example: <button onclick="alert('Clicked!')">Click Me</button>

10. Write the full form of DOM and give its purpose in Web Development.

- DOM: Document Object Model.

- Purpose: Represents a webpage as a tree of objects, allowing JavaScript to manipulate content dynamically.

Example: Changing text: document.getElementById("text").innerText = "Hello World!";

11. What is Dynamic document in Java Script?

A dynamic document is a webpage that updates content, structure, or style in real-time using JavaScript and the DOM, without reloading.

Example: Updating a paragraph's text on button click.

12. What CSS property controls the Stacking Order of Elements?

The z-index property controls the stacking order; higher values place elements on top.

Example: div { position: absolute; z-index: 10; }

Section B (5 Marks Each)

13. Explain the <table> tag in HTML.

The <table> tag creates a table to display data in rows and columns. It uses:

  • <tr>: Table row.
  • <th>: Header cell.
  • <td>: Data cell.

Example:

<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>22</td>
        <td>New York</td>
    </tr>
</table>
14. Discuss the structure of a well-formed XML document.

A well-formed XML document must have:

  • One root element, e.g., <root>.
  • Properly nested tags.
  • All tags closed.
  • Case-sensitive tags.
  • A prolog, e.g., <?xml version="1.0" encoding="UTF-8"?>.

Example:

<?xml version="1.0" encoding="UTF-8"?>
<library>
    <book>
        <title>Learning XML</title>
        <author>Jane Doe</author>
    </book>
</library>
15. Explain CSS Box model in detail.

The CSS Box Model includes:

  • Content: Innermost part (width, height).
  • Padding: Space between content and border.
  • Border: Surrounds padding.
  • Margin: Outer space.

Total size = content + padding + border + margin.

Example:

div {
    width: 200px;
    padding: 10px;
    border: 2px solid black;
    margin: 20px;
}
16. Describe any five types of operators in JavaScript.

- Arithmetic: +, -, *, /, %.

- Assignment: =, +=.

- Comparison: ==, !=, >.

- Logical: &&, ||, !.

- Bitwise: &, |.

Example: let a = 10, b = 3; console.log(a + b); // 13

17. Describe the DOM Tree structure.

The DOM Tree is a hierarchical structure:

  • Root: <html> element.
  • Children: <head>, <body>.
  • Descendants: <title>, <p>, etc.
  • Text Nodes: Content inside tags.

Example: <html><body><p>Hello</p></body></html>

Section C (10 Marks Each)

18. Discuss the different types of lists in HTML.

- Unordered List (<ul>): Bullet points.

<ul>
    <li>Apples</li>
    <li>Oranges</li>
</ul>

- Ordered List (<ol>): Numbered items.

<ol>
    <li>Step 1</li>
    <li>Step 2</li>
</ol>

- Definition List (<dl>): Terms and definitions.

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
</dl>
19. a) Explain different types of style sheets in CSS.

- Inline CSS: Inside an element, e.g., <p style="color: blue;">.

- Internal CSS: In <head> using <style>.

- External CSS: In a .css file, linked via <link>.

19. b) Discuss any two looping statements in CSS.

CSS does not support looping. Assuming JavaScript:

- For Loop:

for (let i = 1; i <= 3; i++) {
    console.log("Count: " + i);
}

- While Loop:

let i = 1;
while (i <= 3) {
    console.log("Count: " + i);
    i++;
}
20. a) Explain different ways to access Element in JavaScript.

- By ID: document.getElementById("myId").

- By Class: document.getElementsByClassName("myClass").

- By Tag: document.getElementsByTagName("p").

- By Selector: document.querySelector(".myClass").

20. b) Discuss the benefits of using JavaScript for creating dynamic documents.

- Real-time updates via DOM.

- Adds interactivity, e.g., form validation.

- Handles events like clicks.

- Supports asynchronous data fetching.

- Cross-browser compatibility.
